    Editing text in Linux

    To edit a work in linux, you have to open your editor (eg: pico). Type in any text you want! like: MYSELF

    There are commands at the bottom of the screen in pico, such as: ^-which is used as control in linux. So to save your workin linux, you have to type ctrl O

    ctrl X-is used to exit from pico editor

    head -1"filename"-displays the top text of a saved file

    tail -1"filename"-displays the bottom text of a saved file

    cat "filename"-is used to display a saved file

    The filename is the name you used to save your file in pico
